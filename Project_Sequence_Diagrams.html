SƠ ĐỒ TRÌNH TỰ (SEQUENCE DIAGRAM)

Mô tả tương tác giữa các lớp (Servlets, DAOs, Database)

1. Chức năng Đăng nhập (Login)

Quy trình xác thực người dùng và phân quyền truy cập.

            sequenceDiagram
                actor User as Người dùng
                participant View as login.jsp
                participant Servlet as LoginServlet
                participant DAO as UserDAO
                participant DB as SQL Server

                User->>View: Nhập Username/Password
                View->>Servlet: POST /login (form data)
                Servlet->>DAO: getUserByCredentials(user, pass)
                DAO->>DB: SELECT * FROM users WHERE username=?...
                DB-->>DAO: User Record (Data)
                DAO-->>Servlet: User Object
                alt Xác thực thành công
                    Servlet->>Servlet: Tạo Session (set 'user' attribute)
                    Servlet-->>User: Redirect to Dashboard (theo Role)
                else Sai thông tin
                    Servlet-->>View: Gửi thông báo lỗi (request attribute)
                    View-->>User: Hiển thị lỗi trên giao diện
                end

2. Chức năng Điểm danh (Mark Attendance)

Giám sát viên cập nhật trạng thái học sinh trong chuyến đi.

            sequenceDiagram
                actor Monitor as Giám sát viên
                participant Page as attendance.jsp
                participant Servlet as AttendanceServlet
                participant DAO as AttendanceDAO
                participant DB as SQL Server

                Monitor->>Page: Chọn trạng thái (PRESENT/ABSENT)
                Page->>Servlet: POST /attendance?action=update
                Servlet->>DAO: updateStudentStatus(tripId, studentId, status)
                DAO->>DB: UPDATE attendances SET status=?...
                DB-->>DAO: Success
                DAO-->>Servlet: return true
                Servlet-->>Page: Refresh data / Show success toast
                Page-->>Monitor: Hiển thị trạng thái mới cập nhật

3. Chức năng Báo vắng (Absence Notification)

Phụ huynh gửi đơn báo nghỉ cho con.

            sequenceDiagram
                actor Parent as Phụ huynh
                participant Page as absence_form.jsp
                participant Servlet as AbsenceServlet
                participant DAO as AbsenceDAO
                participant DB as SQL Server

                Parent->>Page: Chọn ngày nghỉ & Nhập lý do
                Page->>Servlet: POST /absence (submit)
                Servlet->>DAO: insertAbsence(studentId, date, reason)
                DAO->>DB: INSERT INTO absence_notifications...
                DB-->>DAO: Created
                DAO-->>Servlet: Success
                Servlet->>Servlet: Gửi thông báo tới Admin/Monitor
                Servlet-->>Page: Hiển thị "Đã gửi thành công"
                Page-->>Parent: Xem danh sách trạng thái 'Pending'
